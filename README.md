# Django Ecommerce Project

This is a fully-featured eCommerce platform built using Django. The project includes functionalities like cart management, wishlist, payment gateway integration, and user authentication (login, registration, etc.).

## Features

- **Product Listing**: Displays products with details like price, description, and image.
- **Cart Management**: Add, update, or remove products in the cart.
- **Wishlist**: Save products to a wishlist for future reference.
- **Payment Gateway Integration**: Pay for orders using a payment gateway (e.g., Stripe or Razorpay).
- **User Authentication**: Secure login, registration, and logout functionalities.
- **Admin Panel**: Manage products, categories, and users.
- **Responsive Design**: Works on all devices (mobile, tablet, desktop).

## Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS, Bootstrap
- **Database**: SQLite (default)
- **Payment Gateway**: Razorpay
